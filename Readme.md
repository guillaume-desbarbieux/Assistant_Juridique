# Installer Ollama :
https://ollama.com/

> Dans le dossier Ollama :
Télécharger mistral :
```
ollama pull mistral
```

Vérifier que mistral est disponible
```
ollama list
```

Vérifier que Ollama est en service
```
ollama serve
```
